## Getting Started

Requirement

```bash
Node v22.17.0 or above
```

Start the development server (localhost)

```bash
pnpm install

pnpm run dev
```

Build the project

```bash
 pnpm run build
```

## Stack

- Vite (application build)
- React.js (frontend framework)
- TailwindCSS (styling)
- Tanstack/react-query (api handling)
- React-i18n-next (localization handling)
